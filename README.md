# My Kitchen Stories

A beautiful personal cookbook website for managing and sharing recipes.

## Features
- Browse and search recipes
- Add, edit, and delete recipes
- Upload recipe images
- Password-protected admin functions
- Beautiful, responsive design

## Setup

1. Clone this repository
2. Install dependencies: `npm install`
3. Create a `.env` file with your admin password:
```
   ADMIN_PASSWORD=your_password_here
```
4. Start the server: `npm start`
5. Open http://localhost:3000

## Deployment

This app can be deployed to Railway, Render, or Fly.io.

Set the `ADMIN_PASSWORD` environment variable in your hosting platform.

## Tech Stack
- Node.js + Express
- SQLite database
- Vanilla JavaScript
- Beautiful custom CSS
